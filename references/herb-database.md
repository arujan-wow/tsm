# WoW Herb Database

> Static reference for the daily WoW report. Contains all tradeable herbs, item IDs, wowpricehub URL slugs, farming zones, and yield estimates.

## URL Pattern

```
https://wowpricehub.com/{realm}/item/{ItemName}-{itemId}
```

- `{realm}` = e.g. `us/area-52`, `eu/tarren-mill`
- `{ItemName}` = exact in-game name (spaces preserved)
- `{itemId}` = numeric item ID

## Gold/Hour Formula

```
estimated_gold_per_hour = nodes_per_hour * avg_herbs_per_node * current_price_per_herb
```

Use the midpoint of the ranges above for calculations. For rare herbs (Black Lotus, Frost Lotus, Golden Lotus, Anchor Weed, Nightshade, Felwort, Starlight Rose, Orbinid, Nocturnal Lotus), use the lower end since spawn rates are inconsistent.
