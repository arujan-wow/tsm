# WoW Ore Database

> Static reference for `/ore-scan`. Contains all tradeable ores, item IDs, wowpricehub URL slugs, farming zones, and yield estimates.

## URL Pattern

```
https://wowpricehub.com/{realm}/item/{ItemName}-{itemId}
```

- `{realm}` = e.g. `us/area-52`, `eu/tarren-mill`
- `{ItemName}` = exact in-game name with spaces
- `{itemId}` = numeric item ID

## Gold/Hour Formula

```
estimated_gold_per_hour = nodes_per_hour * avg_ore_per_node * current_price_per_ore
```

Use the midpoint of the ranges above for calculations. For rare spawns (Silver, Gold, Truesilver, Khorium, Titanium, Platinum, Trillium, Elethium, Null Stone, Khaz'gorite), use the lower end since spawn rates are inconsistent.
